(inner) getMediciones(req, res) → {void}

Retrieves all stored measurements, filtered by the provided query parameters.

Handles the GET request to `/mediciones`. Retrieves all measurements from the database
Parameters:
Name Type Description
req Request The HTTP request, containing optional query parameters for filtering: - `lugar` (string): Filter by the location where the measurement was taken. - `tipo_gas` (string): Filter by the type of gas measured. - `desde_hora` (string, date-time): Filter measurements taken after this time. - `hasta_hora` (string, date-time): Filter measurements taken before this time.
res Response The HTTP response, which contains a JSON array of measurement objects if successful.

(inner) postMedicion(req, res) → {void}

Creates a new measurement based on the provided data.

Handles the POST request to `/mediciones`. Submits a new measurement to the server, expecting a JSON object in the request body.
Parameters:
Name Type Description
req Request The HTTP request containing a JSON object with the following properties: - `medida` (number, required): The value of the measurement taken by the sensor. - `lugar` (string, required): The location where the measurement was taken. - `tipo_gas` (string, required): The type of gas being measured. - `hora` (string, date-time, required): The time when the measurement was taken.
res Response The HTTP response indicating the result of the operation.
